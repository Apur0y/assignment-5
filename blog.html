Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model(DOM) is the programing interface where the HTML documents represents as a tree and every element of HTML is known as node

How do you select an element from the DOM?

An element can be select from DOM in few ways :

1. Using "ID" : getElementById("ID")

2. Using "ClassName" : getElementById("ClassName")

3. Using "TagName" : getElementById("TagName")

4.To get single element : querySelector("elementIdentity")

5. To get multiple elements: querySelectorAll()

What is event delegaltion in the context of the DOM, and why is it useful?

Event delegation in HTML is a technique that allows you to handle events on multiple elements using a single event listener on a common ancestor.

How do you manipulate an element's attributes and styles using the DOM?

1. You can change the attributes of an element using the setAttribute method or directly by accessing the attribute.

2. To change the styles of an element, you can use the style property.

3. You can add or remove classes using the classList property.
